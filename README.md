
# Foras Khadra - Opportunities Page

A clean-coded, fully responsive web interface for displaying scholarships, training programs, competitions, and educational opportunities.

## About the Project

This project was developed using HTML, CSS, and JavaScript to build a responsive opportunities listing interface.

The page allows users to browse available opportunities and quickly find relevant results through search and filtering options.

## Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Features

* Responsive layout for desktop, tablet, and mobile devices
* Dynamic opportunity cards rendered with JavaScript
* Search opportunities by title
* Filter opportunities by type
* Filter opportunities by funding status
* Real-time results counter
* Empty state message when no matching opportunities are found
* Mobile navigation menu
* Clean and user-friendly interface

## How It Works

* Opportunity data is stored in a JavaScript array.
* Cards are generated dynamically and displayed on the page.
* Search and filter options update the displayed results instantly.
* Funding status is highlighted using different badge styles.
